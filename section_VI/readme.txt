# Additional processed data has been deposited in Bioshare following this link: https://bioshare.bioinformatics.ucdavis.edu/bioshare/view/Dennis_UCSC_Browser_Files/hsds_brain_evolution/

Deposited data includes:

- Raw morphometric measurements for all zebrafish models.
- Processed Seurat object for all integrated single-cell RNA-seq data for all zebrafish models.
- CSV files with the differentially expressed genes (DEGs) for all identified cell types (DESeq models: mutant vs controls).
