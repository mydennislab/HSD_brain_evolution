## These scripts process published data from the following sources:

- Mouse: ENA project PRJNA637987 from https://www.ncbi.nlm.nih.gov/bioproject/PRJNA637987.
- Zebrafish: GEO project GSE158142 from https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE158142.

The processing script follows the pipelines described in Willsley et al. 2020 (https://pubmed.ncbi.nlm.nih.gov/33497602) and Weinschutz Mendes et al. 2023 (https://pubmed.ncbi.nlm.nih.gov/36933215).
